## 📌 О проекте

Эко-карта Воронежа — это интерактивная онлайн-карта, которая собирает в одном месте всю экологическую инфраструктуру города:

- 💧 **130+ точек** с бесплатной питьевой и технической водой (родники, колонки, источники)
- ♻️ **50+ пунктов** приёма вторсырья (батарейки, бумага, пластик, стекло, металл, электроника, ткань)
- 👕 **Секонд-хенды** и эко-магазины
- 🥤 Отдельная страница **«My Cup, Please»** — заведения, где нальют напиток в вашу кружку

Проект-победитель Инженерного чемпионата 2026 (кейс от СИБУР).

---

## 🛠️ Стек технологий

- **Картография:** Leaflet.js + OpenStreetMap
- **Фронтенд:** HTML + CSS + JavaScript
- **Иконки:** Font Awesome
- **Шрифты:** Google Fonts (Inter)
- **Хостинг:** GitHub Pages

---

## 🧭 Функциональность

- 🔍 **Поиск** по названию и адресу
- 🎯 **Фильтры** по типу, району и времени работы
- 📍 **Геолокация** — точки рядом с вами
- 🥤 **«My Cup, Please»** — отдельная карта для экокафе
- 📱 **Адаптивный дизайн** — работает на компьютере, планшете и телефоне
